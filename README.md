# Calculadora IGTF e IVA - Grupo Punteral

Aplicación web profesional para cálculo de IGTF (3%) e IVA (16%) con diseño responsivo, modo oscuro, persistencia, tasa BCV en tiempo real y generación de PDF.

## ✨ Características

- Cálculo instantáneo con Alpine.js
- Modo claro/oscuro con persistencia
- Tasa del BCV actualizada (API pydolarve)
- Persistencia total del formulario
- PDF con logo e información fiscal
- Footer institucional
- SEO optimizado + CSP básica

## 🚀 Tecnologías

- Alpine.js 3.x
- Tailwind CSS
- jsPDF
- localStorage API
- Fetch API (BCV)
